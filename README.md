Social Media Web Application
This project is a PHP-based social media web application designed to provide users with a platform for sharing posts, following other users, and managing profiles securely.

Features
User Authentication: Secure signup and login functionality.
Post Management: Create, edit, and delete posts with ease.
Like and Follow: Interact with posts by liking and following other users.
Profile Editing: Update and manage user profiles.
Secure Authorization: Ensure safe access with robust authorization mechanisms.
Technologies Used
PHP: Backend server-side scripting.
MySQL: Relational database management system.
HTML/CSS: Frontend markup and styling.
JavaScript: Client-side scripting for dynamic interactions.
Installation
Clone the repository:

bash

Usage
create table in DB example-php then navigate to example-php.sql then import it as sql  .

Explore the various features such as creating posts, interacting with posts, and managing your profile.
